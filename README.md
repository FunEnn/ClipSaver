# ClipSaver - 全局剪贴板管理器

## 应用简介

能够自动记录在 Android 设备上复制的所有文本内容，通过无障碍服务技术，实现全局剪贴板监控。

## 快速开始

### 安装要求

- Android 8.0+ (API 26+)
- 无障碍服务权限
- 通知权限（Android 13+）

**权限配置**
   - 首次启动时，应用会请求必要的权限
   - **重要**：必须手动开启无障碍服务才能使用全局剪贴板功能

### 无障碍服务配置

1. 打开应用后，在历史记录页面顶部会显示"未开启无障碍服务"提示
2. 点击"前往开启"按钮
3. 在系统设置中找到 "ClipSaver" 并开启服务
4. 返回应用，服务将自动启动

## 开发环境

### 系统要求

- Android Studio 25.1
- Android SDK 36

## 功能特性

### 剪贴板管理
- 自动记录所有复制操作
- 实时剪贴板内容监控
- 支持文本内容筛选
- 历史记录搜索功能
